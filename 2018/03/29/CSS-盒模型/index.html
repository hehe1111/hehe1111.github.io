简介 CSS 盒模型

盒模型

在一个文档中，每个元素都被表示为一个矩形的盒子。确定这些盒子的尺寸、属性（包括边框、背景、颜色等）和位置是渲染引擎的目标。每个盒子均有 4 部分组成：

  • 内容区（有 width、height 属性）：包含元素真实内容的区域。
  • padding（内边距）：背景、颜色等会延伸到 padding。
  • border（边框）：可以用来画小三角形等简单图形。
  • margin（外边距）：存在外边距合并的情况。
    • 三种情况有可能会出现外边距合并
      • 相邻兄弟元素
      • 父元素与第一个或最后一个子元素之间，子元素的外边距溢出父元素
      • 空的块级元素
    • 以下几个属性加在父元素上，可以清除父子外边距合并。当然，清除外边距合并还有其他方法。
      • overflow: hidden;
      • padding-top: 0.1px;
      • display: flow-root;
    • float 或绝对定位（absolute、fixed）的元素的外边距不会被合并。

定义不同的盒模型

  • box-sizing: border-box：布局比较方便，计算时只需要考虑 width + margin-left + margin-right。真实内容宽度为 width 自动减去 padding 和 border。
    • width = content-width + padding-left + padding-right + border-left + border-right
    • height = content-height + padding-top + padding-bottom + border-top + border-bottom
  • box-sizing: content-box：真实内容宽度为 width，计算时要考虑 padding、border、margin。

其他

  • 对于非替换元素的行内元素，其占用空间有 line-height 决定。
  • 行内元素默认是基线对齐：vertical-align: baseline。
  • 可替换元素：外观渲染独立于 CSS 的外部对象。
    • <img>、<video>、<textarea>、<input> 等
  • <img>、<input>等标签属于替换元素，一般都有内在尺寸和宽高比，可以设定 width 和 height。
