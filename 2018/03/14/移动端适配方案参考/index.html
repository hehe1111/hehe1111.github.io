页面在移动端的一些适配方案

在 <head> 标签中，引入以下 <meta> 标签

该 <meta> 标签用于将

  • 页面宽度调整为用户设备宽度（width=device-width）
  • 禁止用户手动缩放（user-scalable=no）
  • 初始页面缩放比例、最大、最小页面缩放比例均为 1，即不缩放。

media query - 媒体查询

媒体查询能够根据用户设备屏幕的尺寸或者保持在横向或者纵向的模式时，提供不同的样式，从而可以实现在不改变内容的前提下，呈现不同的表现风格，使页面保持美观。

动态 rem 方案

rem的作用

  • rem 作用于非根元素时，相对于根元素字体大小；
  • rem 作用于根元素的 font-size 属性时，相对于初始字体大小（浏览器默认字体大小，通常是 16px，chrome 是 12px）
  • rem 布局的本质是等比缩放

使用 SCSS：在 SCSS 里使用 PX2REM

相比上一个方案的优点：可以不用自己计算 rem 值，直接写 px 值即可。

定义好的 px 函数会将传进去的 px 值转成 rem 值。

之后将 scss 文件转成 css 文件，再在 HTML 页面引入 css 文件即可。
