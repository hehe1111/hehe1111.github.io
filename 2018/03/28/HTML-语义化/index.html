HTML 语义化

发表于 2018-03-28 | 更新于 2018-06-14 | 分类于 前端
用正确的标签做正确的事
  1. 使用的标签要能够适当的反应出标签内容是什么，而不是满屏都是 <div> 标签。<div> 本身并没有任何语义，从英文（division）角度看，它仅是代表了一个分区，但这个分区的内容是什么并无法判断。使用语义化标签，可以更清晰地表达文档结构。例如：页眉可以用 <header>，边栏可以用 <aside>，导航栏可以用 <nav>，底部页脚可以用 <footer>，标题用 <h1> ~ h6，段落用 <p>，文章用 <article> 等等。
  2. 合理使用合适的标签，能够使 HTML 代码易读，有利于维护。
