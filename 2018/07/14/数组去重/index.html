数组去重

发表于 2018-07-14 | 更新于 2018-07-26 | 分类于 前端
去掉数组里相同的元素

第一种方法的缺点：

  • 只能用于去重数值，非空字符串，数组，空对象元素，null，undefined
  • 不能区分非空对象（如{"1":1}, {"2":2}）
  • 不能区分布尔值（true/false）和字符串形式的”true”/“false”
  • 不能区分数值1和字符串”1”
  • 不能识别 NaN 和 “NaN”
  • 不能区分字符串"null"，"undefined"

第二种方法的缺点：

  • 不能区分对象（Object Array等），故简单类型（number string boolean null undefined symbol）优先选[...new Set(array)]。
  • 兼容性差。

第三种方法：结合以上两种方法

  • 在 ES6 提供的 [...new Set(array)] 的基础上，先对数组进行第一步去重。问题在于，这种方法不能区分对象（Object Array等）。对于 Object 和 Array，可以用 JSON.stringify() 方法结合第一种方法来进行去重。结果表明，第三种方法去重效果优于前两种。
  • 缺点：要进行两次去重。
  • 兼容性差。
