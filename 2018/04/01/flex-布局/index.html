flex 布局属性简介

Flex 是 Flexible Box 的缩写，意为”弹性布局”。
设为 Flex 布局以后，子元素的 float、clear 和 vertical-align 属性将失效。

  • flex 属性分为容器与项目属性

容器属性

  1. flex-direction: 决定主轴方向。默认值 row。
  2. flex-wrap: 决定是否换行。默认值 nowrap，不换行。
  3. flex-flow: 前面两个属性的简写。默认值 row nowrap。
  4. justify-content: 决定主轴的对齐方式。默认值 flex-start。
  5. align-items: 决定侧轴的对齐方式。默认值 stretch，占满整个容器的高度。
  6. align-content: 用于多轴线对齐，单轴线时不起作用。默认值 stretch。

项目属性

  1. order: 决定项目的顺序。数值越小越靠前，默认值 0，可以为负值。
  2. flex-grow: 决定项目的放大比例。默认值 0，即“不放大”。
  3. flex-shrink: 决定项目的缩小比例。默认值 1，即“有必要时会缩小”。
  4. flex-basis: 决定项目占据的主轴空间。默认值 auto，即“项目原本的大小”。可以用 px 等单位值设置为固定大小。
  5. flex: 以上三个属性的简写。默认值 0 1 auto，即“不放大，默认为原本的大小，有必要时会缩小”。
  6. align-self: 可决定单个元素的侧轴对齐方式，覆盖 align-items 的值。
